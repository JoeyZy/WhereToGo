How does Component work?

Component is constructed upon the existing DOM. To do so, it looks for an element with attribute data-model (see {@link Component.MODEL_ATTR}) set to the id value passed as an argument to a constructor. If this element has children with model-ids (hereafter, a model-id is a value of the data-model attribute) in the form <model-id of the parent>.<model-id>, these elements get stored and observed.

For example, given the following markup:

    <form data-model="userForm">
        Name: <input type="text"   data-model="userForm.name"><br/>
        Age : <input type="number" data-model="userForm.age" ><br/>
    </form>

you may construct a view like so:

    var userView = new Component("userForm", userModel);

Now userView has the following properties:

Property Value (DOM element)
userView.element form[data-model=userForm]
userView.elements["userForm.name"] input[data-model=userForm.name]
userView.elements["userForm.age"] input[data-model=userForm.age]

To get (or set) the value of one of component's elements, you may call method elementValue(), e.g.

    userView.elementValue("userForm.name"   ); // get the value of "Name" input
    userView.elementValue("userForm.age", 25); // set the value of "Age"  input
