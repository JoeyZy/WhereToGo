The domain-specific representation of the information on which the application operates.

The domain-specific representation of the information on which the application operates. The model is another name for the domain layer. This is the value you send with AJAX calls; based on this value, you update your view.

The Model is an implementation of Observable: Adds val() method for updating the value of the model
